Time Entry Components:

Time Entry has the following components:
  • id: number. This number will be automatically added.
  • title. It's the name of the item.
  • description. Description of the item, referring to the work done and for whom.
  • startTime. Choose from dropdown menu the following Items:
    • Current Time: Current local time.
    • Today Start Time Standard: Date of today @ 08:30am.
    • Today Stop Time Standard : Date of today @ 17:00pm.
    • Custom TimeDate: Custom Date & Time.
